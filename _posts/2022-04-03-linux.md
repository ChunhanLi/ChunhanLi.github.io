### 命令

- grep

grep命令用于查找文件里符合条件的字符串。

```
grep test *file 查找后缀有file字样包含test字符串的
grep -r test /etc/acpi 以递归的方式查找符合条件的文件
grep -v 查找不符合条件的行
grep -i 不区分大小写
grep -n 显示行数
grep -c 只显示匹配行的计数 
grep -h 不显示文件名
grep -l 只显示文件名
grep -e 正则表达式

grep -C 5 pattern file 显示file文件里匹配pattern的上下5行
grep -B 前5行
grep -A 后5行

grep pattern1|pattern2 file 或
grep pattern file | grep pattern2 且
```

- 正则表达式

```
^ 以什么开头
$ 以什么结尾
. 匹配一个字符
* 匹配0个或多个
[] 匹配其中的单个字符
[^] 匹配不在集合里的
[-] A-Z范围里的
\{m\} 重复字符m次
\{m,\} 重复字符至少m次
\{m,n\}  m-n行
+ 重复前一个字符一个或一个以上
| 或多个字符串 g(la|oo)d glad good都能出
() 分组过滤匹配
```
